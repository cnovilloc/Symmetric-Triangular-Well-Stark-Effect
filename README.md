# Symmetric-Triangular-Well-Stark-Effect
Numerical implementation for studying the Stark effect in the symmetric triangular quantum well using perturbative and semiclassical methods.

## Contents

- `CodeEnergyCorrections.ipynb`  
  Perturbative calculations using Rayleigh–Schrödinger and Dalgarno–Lewis methods, together with exact energy corrections.

- `CodeWKB.ipynb`  
  WKB approximation and semiclassical calculations.

## Requirements

```bash
numpy
scipy
matplotlib
sympy
jupyter
```

## Usage

Run the notebooks with Jupyter:

```bash
jupyter notebook
```

## Notes

The calculations use normalized units with

\[
m = 0.5, \qquad \hbar = 1, \qquad F = 1.
\]
